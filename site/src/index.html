Graphic Design in WebAssembly

gdl is a high-performance WebAssembly graphic design library that runs both natively and on the web.

Features

Unlike other libraries, this library is built with 100% pure Rust, so security and safety is guaranteed.

  • Web-Assembly Friendly

    For web-based graphic design, gdl is 4-10x faster than JS, leading to faster results, and less lag.

  • Over 90 functions

    gdl provides functions for every domain of graphic design.

How it works

Write once, run anywhere.

gdl can be executed:

  • In the browser via JS
  • On Node.JS
  • Natively on any machine

gdl compiles to WebAssembly, which allows the execution of native-speed code in the browser and on Node.JS. This means incredible performance and since gdl's core library is written in Rust, it promises security and safety.
