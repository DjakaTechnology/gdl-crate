Graphic Design in WebAssembly

Metallic is a high-performance WebAssembly graphic design library that runs both natively and on the web.

Features

Unlike other libraries, this library is built with 100% pure Rust, so security and safety is guaranteed.

  • Web-Assembly Friendly

    For web-based graphic design, Metallic is 4-10x faster than JS, leading to faster results, and less lag.

  • Over 90 functions

    Metallic provides functions for every domain of graphic design.

    Read More

  • Call WASM with JS

    This library's WASM functions can be called via JS, allowing for zero-cost abstraction and faster development.

    Read More

How it works

Write once, run anywhere.

Metallic can be executed:

  • In the browser via JS
  • On Node.JS
  • Natively on any machine

Metallic compiles to WebAssembly, which allows the execution of native-speed code in the browser and on Node.JS. This means incredible performance and since Metallic's core library is written in Rust, it promises security and safety.
