Graphic Design with Rust

gdl is a high-performance Rust graphic design library that runs both natively and on the web.

Features

Unlike other libraries, this library is built with 100% pure Rust, so security and safety is guaranteed.

  • Web-Assembly Friendly

    For web-based graphic design, gdl is 4-10x faster than JS, leading to faster results, and less lag.

    Read More

  • Over 90 functions

    gdl provides functions for every domain of graphic design.

    Read More

  • Call WASM with JS

    This library's WASM functions can be called via JS, allowing for zero-cost abstraction and faster development.

    Read More

How it works

Create 2D Graphics

gdl allows you to create social media graphics, product mockups, and over 10 other types of graphic.

  • Pair gdl with its sibling library, Photon, and apply image effects or filters to your final designs or images.
  • Low-level control: gdl gives you fine-tuned control over your designs.
    • View all [functions here](https://silvia-odwyer.github.io/gdl/docs/gdl/all.html).

Built-in Data Visualisation

gdl gives you the tools to create barcharts, histograms, and other data visualisation charts, allowing you to visualise data and create graphics or even infographics based on your data.

15+ Built-in Templates

Kickstart your designs with over 15 built-in templates. Just add text, and GDL will handle the rest. You can also create collages and combine images together to create interactive graphics. More coming soon!

Run GDL anywhere.

GDL can be executed:

  • In the browser via JS
  • On Node.JS
  • Natively on any machine

GDL compiles to WebAssembly, which allows the execution of native-speed code in the browser and on Node.JS. Since it is written in Rust, it can also be executed natively.

Photon's core library is a Rust crate, available from Cargo.

Write code in JS without a single line of Rust.

JS code gets exported to WebAssembly.

Get Started In Just 3 Lines of Code

You can create a design in just 3 lines of code; the library is built with zero-cost abstraction in mind.

You can view the examples for more code.

                      extern crate gdl;
  use gdl::{presets};

  fn main() {
    let img = open_image("image.JPG");
    
    let postcard = presets::postcard(&mut img, "Welcome", "to Paris");
    
    save_image(postcard, "new_image.PNG");
  }
